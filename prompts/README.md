# Prompt registry

Owner: AI Governance. Mỗi prompt đặt trong thư mục theo capability và lưu bằng
tên phiên bản `vN.md`, `vN.N.md` hoặc `vN.N.N.md`. Không sửa nội dung một phiên
bản đã phát hành; tạo phiên bản mới và cập nhật release manifest/eval evidence.

Các thư mục hiện tại là scaffold cho Chat sau, chưa có prompt production.
